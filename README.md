# OPEN MOD - A OPEN SOURCE FPS SDK

## ABOUT

OpenMod SDK Is a Mod Template built over MapBase, the goal of this mod is allow mod makers make their own mod with some nice perks.

## RUNNING

To run the mod you need:

* Source SDK 2013 SP
  * Set this to the upcoming branch on the game settings > betas tab
* **(Optional)** Half-life 2 episodes + lost coast
